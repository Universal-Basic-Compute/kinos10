# Progress Memories

This directory stores information about the user's therapeutic progress, milestones, and growth journey.

## Purpose
To maintain a continuous record of the user's therapeutic development, allowing TherapyKin to recognize, reinforce, and build upon progress over time.

## Content Types

### Milestone Records
Documentation of significant therapeutic achievements, breakthroughs, or insights.

### Skill Development
Tracking of specific therapeutic skills the user has learned, practiced, and integrated.

### Pattern Changes
Evidence of shifts in thought patterns, emotional responses, or behavioral habits.

### Goal Progress
Updates on movement toward user-identified therapeutic goals.

### Setback Navigation
Information about how the user has managed challenges or setbacks in their journey.

## Usage Guidelines

- Reference progress memories to reinforce growth and resilience
- Use milestone records to help the user recognize their own development
- Connect current challenges to previously navigated difficulties
- Identify skill development patterns to suggest appropriate next steps
- Update progress memories after significant therapeutic work
- Maintain a balanced view that acknowledges both progress and ongoing challenges

## Example Memory Structure

```
progress/
  ├── milestones/
  │   ├── first_boundary_setting.txt
  │   ├── panic_management_breakthrough.txt
  │   └── grief_processing_milestone.txt
  │
  ├── skills/
  │   ├── thought_challenging.txt
  │   ├── mindfulness_practice.txt
  │   └── assertive_communication.txt
  │
  ├── goals/
  │   ├── reduce_anxiety.txt
  │   ├── improve_relationships.txt
  │   └── process_grief.txt
  │
  └── patterns/
      ├── emotional_regulation.txt
      ├── self_talk.txt
      └── behavioral_responses.txt
```

This directory structure will be populated as the therapeutic relationship develops, creating a rich record of the user's journey.
