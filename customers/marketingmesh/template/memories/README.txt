# MarketingMesh Memories

This directory contains structured information about marketing campaigns, personas, and analysis results. The JSON files stored here serve as the system's memory, allowing for consistent reference and evolution of marketing strategies over time.

## Directory Structure

- `/business/` - Information about the business, its products, and positioning
- `/personas/` - Customer personas and audience segments
- `/campaigns/` - Marketing campaign structures and assets
- `/performance/` - Historical performance data and insights

## File Format

Most files in this directory use a structured JSON format to enable systematic processing and integration across the MarketingMesh platform. These structures allow for:

1. Consistent organization of marketing information
2. Easy retrieval of specific marketing elements
3. Systematic analysis of marketing performance
4. Evolution of marketing strategies based on results

The system will automatically generate and update these files as you work with MarketingMesh.
