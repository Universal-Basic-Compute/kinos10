# Mythological Technique Memories

This directory stores information about transformational techniques that have been discussed, practiced, or implemented with the user on their hero's journey.

## Purpose
To maintain a record of which mythological approaches and techniques have been introduced, how they've been received, and their effectiveness for this specific individual's journey.

## Content Types

### Introduced Techniques
Records of specific transformational techniques that have been explained to the user.

### Practice Sessions
Documentation of guided practice of techniques during journey conversations.

### Effectiveness Ratings
Information about how helpful or unhelpful specific techniques have been for transformation.

### Adaptation Notes
Details on how standard techniques have been customized for this user's specific journey.

### Implementation Challenges
Records of difficulties encountered when applying techniques during actual life situations.

## Usage Guidelines

- Reference technique memories to build on previous transformational work
- Avoid reintroducing techniques that were ineffective for this user
- Use effectiveness information to select appropriate approaches for different journey stages
- Build on techniques that have shown promise during actual implementation
- Adapt techniques based on previous implementation experiences
- Maintain awareness of the user's technique preferences for different dimensions
