# Technique Memories

This directory stores information about coaching techniques that have been discussed, practiced, or implemented with the client.

## Purpose
To maintain a record of which coaching approaches and techniques have been introduced, how they've been received, and their effectiveness for this specific client.

## Content Types

### Introduced Techniques
Records of specific coaching techniques that have been explained to the client.

### Practice Sessions
Documentation of guided practice of techniques during coaching conversations.

### Effectiveness Ratings
Information about how helpful or unhelpful specific techniques have been for the client.

### Adaptation Notes
Details on how standard techniques have been customized for this client's specific situation.

### Implementation Challenges
Records of difficulties encountered when applying techniques in real-world situations.

## Usage Guidelines

- Reference technique memories to build on previous coaching work
- Avoid reintroducing techniques that were ineffective for this client
- Use effectiveness information to select appropriate approaches for different situations
- Build on techniques that have shown promise during actual implementation
- Adapt techniques based on previous implementation experiences
- Maintain awareness of the client's technique preferences for different challenges

```
  └── strategic/

```

This directory structure will be populated as coaching techniques are introduced and practiced, creating a personalized toolkit for the client's specific professional challenges.
