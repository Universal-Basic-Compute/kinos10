# SOTTOSIMULATION SYSTÈME

Ce répertoire contient les fichiers de configuration système :

FICHIERS PRINCIPAUX :
- map.json              - Mapping des composants et transitions
- simulation_config.txt - Configuration générale simulation
- simulation_engine.txt - Définition moteur simulation
- simulation_patterns.txt - Patterns de communication

INTÉGRATION SOTTO :
1. Communication
   - Envoi commandes vocales
   - Réception réponses
   - Mise à jour contexte
   - Déclenchement actions

2. Synchronisation
   - Events temps réel
   - État simulation
   - Contexte restaurant
   - Feedback performance

3. Validation
   - Logs communication
   - Métriques réponse
   - Analyse qualité
   - Détection erreurs

UTILISATION :
1. Configuration
   - Ajuster paramètres dans simulation_config.txt
   - Définir patterns dans simulation_patterns.txt
   - Configurer flows dans map.json

2. Monitoring
   - Surveiller logs communication
   - Vérifier métriques performance
   - Analyser patterns erreurs

3. Maintenance
   - Mettre à jour patterns communication
   - Optimiser paramètres simulation
   - Ajuster règles transition
