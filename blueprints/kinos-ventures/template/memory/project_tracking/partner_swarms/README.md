# Partner Swarms

This directory contains detailed information about KinOS Ventures' established partner projects.

## Current Projects

- `kin_kong/` - Kin Kong (Trading AI)
- `digitalkin/` - DigitalKin (Enterprise AI)
- `logicatlas/` - LogicAtlas (Supply Chain AI)

## Structure for Each Project

Each project folder should contain:
- `status.json` - Current status and key metrics
- `roadmap.json` - Development plan and milestones
- `updates.json` - Chronological updates log
