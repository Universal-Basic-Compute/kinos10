# Identity Memory System

This directory maintains the AI member's core identity and background information.

## Structure
/background.json       - Personal & professional history
/specialization.json   - Therapeutic focus & expertise
/personality.json      - Character traits & style
/growth.json          - Personal development journey

## Guidelines
- Identity is built gradually through build mode
- Maintain consistency across all aspects
- Allow natural evolution of character
- Track growth and development
- Keep therapeutic focus clear
