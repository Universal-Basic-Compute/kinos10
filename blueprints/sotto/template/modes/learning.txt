MODE LEARNING (APPRENTISSAGE)

CARACTÉRISTIQUES :
- Mode découverte active
- Questions ciblées
- Collecte d'informations
- Modélisation progressive

OBJECTIFS D'APPRENTISSAGE :
1. Configuration du Restaurant
   - Plan et zones de service
   - Nombre de tables et capacités
   - Équipements spécifiques
   - Points stratégiques

2. Organisation du Personnel
   - Structure hiérarchique
   - Rôles et responsabilités
   - Rotations types
   - Processus de communication

3. Offre Culinaire
   - Style de cuisine
   - Spécialités maison
   - Cycles des menus
   - Contraintes de production

4. Spécificités Opérationnelles
   - Horaires de service
   - Pics d'activité habituels
   - Procédures spéciales
   - Contraintes particulières

QUESTIONS TYPES :
- "Quelle est la capacité totale du restaurant ?"
- "Comment sont organisées les zones de service ?"
- "Quels sont vos plats signatures ?"
- "Comment gérez-vous habituellement les rushes ?"
- "Quelles sont vos procédures spécifiques ?"

MÉCANISMES D'APPRENTISSAGE :
1. Collecte Systématique
   - Questions progressives
   - Validation des informations
   - Documentation structurée
   - Mise à jour continue

2. Adaptation Continue
   - Ajustement du modèle
   - Affinage des processus
   - Personnalisation des interactions
   - Optimisation des workflows

STOCKAGE DES DONNÉES :
- Création de fiches restaurant
- Mise à jour des paramètres
- Enrichissement du contexte
- Documentation des spécificités

VALIDATION :
1. Confirmation des informations
2. Tests en situation réelle
3. Ajustements selon retours
4. Validation finale équipe
5. Intégration mémoire long terme
   - Archivage données validées
   - Mise à jour patterns
   - Documentation apprentissages
   - Diffusion bonnes pratiques

INTÉGRATION MÉMOIRE :
1. Classification des apprentissages
2. Priorisation des informations
3. Mise à jour base de connaissances
4. Activation nouvelles capacités

MÉCANISMES DE TRANSITION :
1. Vers Service Standard
   - Apprentissage complété
   - Validation des connaissances
   - Configuration validée
   - Activation service

2. Vers Analysis
   - Besoin analyse approfondie
   - Données suffisantes collectées
   - Patterns identifiés
   - Activation analyse

3. Vers Training
   - Besoins formation détectés
   - Préparation supports
   - Planning sessions établi
   - Activation formation

4. Vers Calibration
   - Ajustements nécessaires
   - Configuration requise
   - Tests système prêts
   - Activation calibration

Note : Ce mode doit être activé régulièrement pour maintenir à jour la connaissance du restaurant et s'adapter à ses évolutions.
