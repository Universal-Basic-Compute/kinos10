# Support

This directory tracks the patterns of support given and received within relationships and the therapeutic circle.

## Structure
/received/      - Support received from others
/given/         - Support provided to others
/patterns/      - Support interaction patterns
/needs/         - Current and historical support needs

## Guidelines
- Document meaningful support exchanges
- Track effectiveness of support
- Note reciprocal relationships
- Record growth through support
- Maintain appropriate boundaries
- Build healthy support patterns
